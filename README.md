# Deploy backend

## DATABASE

Install Docker

in dir /backend

```
docker-compose up -d
```

## Dependencies

```
npm install
```

## Run dev mode

```
npm run start:dev
```

# Deploy frontend

## Dependencies

```
npm install
```

## Run dev mode

```
npm run serve
```

## Run production mode

```
npm run build
```
